# Development mode

1. Clone the repository
2. Execute

```
yarn install
```

3. Have Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Up the data base

```
docker-compose up -d
```

5. Rebuild data base with the seed

```
http://localhost:3000/api/v2/seed
```

## Stack used

- MongoDB
- Nest.js
